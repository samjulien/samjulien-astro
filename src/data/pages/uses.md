This is my [uses page](https://uses.tech), a list of hardware, software, and tools I use on a daily basis. I'm always refining my setup to optimize for productivity and ergonomics.

## General hardware

- [Moonlander Keyboard](https://www.zsa.io/moonlander) - Ergonomic split mechanical keyboard by ZSA
- [Autonomous White L-Shaped Standing Desk](https://www.autonomous.ai/standing-desks/autonomous-desk-duospace) - Ergonomic height-adjustable corner desk
- [Kensington Expert Mouse Trackball](https://www.kensington.com/p/products/electronic-control-solutions/trackball-products/expert-mouse-wireless-trackball-2/) - Ergonomic wireless trackball mouse
- [Herman Miller Aeron Chair](https://www.hermanmiller.com/products/seating/office-chairs/aeron-chair/) - Ergonomic office chair

## Development tools

I write code daily in JavaScript and Python. This site is built with [Astro](https://astro.build). In addition to the usual tools like git, here are my favorite coding tools:

- [Cursor](https://www.cursor.com/en) - AI-powered code editor, mostly used at work
- [Windsurf](https://codeium.com/windsurf) - AI agentic code editor by Codeium, mostly used for side projects
- [Working Copy](https://workingcopy.app/) - Git client for iOS
- [Drafts](https://getdrafts.com/) - Text editor and automation tool for iOS and Mac
- [LM Studio](https://lmstudio.ai/) - Application for running large language models locally
